2) Unzip and move to C:\
3) Open CMD with admin permissions

4) cd C:\kafka_2.12-2.3.0\bin
4.1) zookeeper-security-migration.bat ..\..\Config\zookeeper.properties
4.1) kafka-server-start.bat ..\..\Config\server.properties
